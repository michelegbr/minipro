# EduMaster - Learning Management System

EduMaster adalah aplikasi Fullstack Web Development untuk manajemen pembelajaran, dibangun menggunakan React dan Laravel.

## Live Links
- Frontend: https://edumaster-gamma.vercel.app/
- Backend API: https://38.47.180.18:8443/student13/api

## Repository Structure
- `/frontend` : Source code React & Vite
- `/backend`  : Source code Laravel 11

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Laravel 11, MySQL
- Deployment: Vercel (Frontend), VPS Ubuntu Server (Backend)
